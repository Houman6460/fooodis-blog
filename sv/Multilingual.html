​Förbättra din restaurangs räckvidd med flerspråkiga menyer

​Step-by-Step Guide to Adding and Managing Multiple Languages

​Utöka din kundbas och förbättra matupplevelsen genom att erbjuda din restaurangs meny på flera språk. Den här guiden hjälper dig att aktivera flerspråkigt stöd och smidigt översätta din meny och restauranginformation.

Guide för att skapa en flerspråkig restaurangwebbplats

Aktivera flerspråkigt stöd
1. Gå till sektionen Appar
  ○ Navigera till "Appar" från sidofältet på din restaurangs dashboard.

2. Aktivera Google Translate
  ○ Klicka på "Google Translate".

3. Markera följande alternativ
  ○ Visa Google Translate.
  ○ Visa Alla Språk.
  ○ Lista de språkkoder du vill visa (t.ex. "en, sv" för engelska och svenska).

4. Spara ändringarna
  ○ Klicka på "Spara" för att bekräfta inställningarna.

Översätta menyer och artiklar till nya språk

1. Växla till det nya språket
  ○ Klicka på språkrullgardinsmenyn och välj det nyligen tillagda språket (t.ex. tyska).

2. Översätt kategorier
  ○ Klicka på en kategori för att redigera dess namn.
  ○ Ange det översatta kategorinamnet på det nya språket.

3. Översätt artiklar
  ○ Klicka på en artikel för att öppna dess detaljer.
  ○ Ange det översatta artikelnamnet, beskrivningen och andra detaljer.

4. Spara ändringarna
  ○ Klicka på Spara för att säkerställa att ändringarna tillämpas.

Översätta restauranginformation till nya språk

1. Översätta restauranginformation
  1.1 Gå till Restauranghantering
      ○ Navigera till sektionen Restaurang från sidofältet på din dashboard.
  1.2 Redigera restauranginformation
      ○ Växla till det nyligen tillagda språket.
      ○ Ange översatta detaljer för restaurangens namn, beskrivning, adress och annan information.
  1.3 Spara ändringarna
      ○ Klicka på Spara för att bekräfta ändringarna.

2. Ändra standardspråk
  ○ Klicka på språkrullgardinsmenyn och välj Gör [Språk] Standard.

3. Ytterligare tipsTa bort ett språk
  ○ Klicka på Ta bort [Språk] i språkrullgardinsmenyn om ett specifikt språk inte längre behövs.

​Vanliga frågor om flerspråkiga menyer

​Denna FAQ-sektion är utformad för att ge tydliga svar på vanliga frågor om att ställa in och hantera flerspråkiga menyer och restauranginformation. Använd dessa frågor och svar för att säkerställa en smidig översättningsprocess och maximera din räckvidd med Fooodis.com:s omfattande flerspråkiga funktioner.

​Hur aktiverar jag flerspråkigt stöd för min restaurangwebbplats?

​För att aktivera flerspråkigt stöd, gå till sektionen Appar på din dashboard, välj Google Translate och aktivera Visa Google Translate samt Visa Alla Språk. Klicka på Spara för att bekräfta.

​Hur lägger jag till ett nytt språk i min meny?

​I sektionen Meny på din dashboard, klicka på språkrullgardinsmenyn, välj Lägg till nytt språk, välj önskat språk och klicka på Lägg till nytt språk.

​Hur översätter jag kategorier och menyartiklar?

​Växla till det nya språket i rullgardinsmenyn, klicka sedan på en kategori eller menyartikel för att redigera och ange det översatta namnet och beskrivningen.

​Kan jag översätta restauranginformation som beskrivning och adress?
  • –
  • –
    ​Ja, gå till sektionen Restaurang på din dashboard, växla till önskat språk och ange översatta detaljer för restaurangens namn, beskrivning och adress.
​Hur ändrar jag standardspråket för min restaurangwebbplats?

​Klicka på språkrullgardinsmenyn och välj Gör [Språk] Standard för att ställa in ett specifikt språk som standard.

​Är det möjligt att ta bort ett språk efter att det har lagts till?

​Ja, välj språket från rullgardinsmenyn och klicka på Ta bort [Språk] om det inte längre behövs.

​Kan jag massuppladda översatta menyartiklar?

​Ja, du kan använda funktionen Importera från CSV för att massuppladda översatta menyartiklar.
