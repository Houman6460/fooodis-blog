​Registrera din restaurang på Fooodis.com

Kom igång med din restaurangs online-närvaro

​Att registrera din restaurang på Fooodis.com är det första steget mot att hantera dina verksamheter digitalt. I denna sektion guidar vi dig genom:
Hitta registreringslänken på hemsidan.
Fyll i viktiga detaljer som din restaurangs namn och ägarinformation.
Skicka in ditt registreringsformulär.
När registreringen är klar får du ett e-postmeddelande med dina inloggningsuppgifter, inklusive ett automatiskt genererat lösenord. Med denna information kan du komma åt din dashboard och börja anpassa din restaurangs digitala upplevelse.

Registrera din restaurang på Fooodis.com

Steg-för-steg-guide för att påbörja din registrering:

1. Gå till registreringssidan
​   ○ På startsidan av Fooodis.com, hitta "Registrera" -länken i menyn längst upp till höger på skärmen (1). 
​   ○ Genom att klicka på denna länk kommer du att omdirigeras till registreringssidan.

2. Ange information om din restaurang

​   ○ På sidan Registrera din restaurang (2) fyll i de obligatoriska fälten:
   ○ Restaurangens namn: Ange namnet på din restaurang.
   ○ Ägarens namn: Ange ditt fullständiga namn.
   ○ Ägarens e-post: Ange din e-postadress.
   ○ Ägarens telefon: Ange ditt kontaktnummer.

3. Spara informationen
​​   ○ Efter att ha fyllt i detaljerna, klicka på Spara-knappen längst ned på registreringsformuläret (3).
​   ○ Du kommer att få ett e-postmeddelande med inloggningsuppgifterna för din restaurang, inklusive ett automatiskt genererat lösenord (observera: lösenordet är inte skiftlägeskänsligt).

​Åtkomst till din restaurangdashboard

Logga in och hantera din restaurangs dagliga verksamhet

​Efter att du har slutfört din registrering får du tillgång till Fooodis.com:s kraftfulla dashboard, där du kan hantera din restaurangs menyer, beställningar och mycket mer.Den här delen av guiden förklarar hur du loggar in på ditt konto med de inloggningsuppgifter som skickades till dig via e-post. Vi täcker även hur du kan komma åt inloggningssidan när som helst från startsidan, så att du kan hålla dig ansluten till din restaurangs verksamhet.När du har loggat in kan du göra viktiga justeringar och förbättringar för att stärka din restaurangs prestation.

Åtkomst till din restaurangdashboard

Hur du loggar in och hanterar din restaurang:

4. Åtkomst till din dashboard
  ○ När ditt konto har skapats kan du logga in på Fooodis-plattformen för att komma åt din restaurangs dashboard (4). Här kan du hantera din meny, beställningar och andra operativa inställningar.

5. Logga in när som helst via startsidan
  ○ För att logga in när som helst, klicka på knappen "Logga in" på startsidan på Fooodis.com (5). Detta tar dig till inloggningssidan.

6. Använd dina inloggningsuppgifter
  ○ På inloggningssidan (6), ange ditt användarnamn (e-postadressen du registrerade dig med) och lösenordet du fick via e-post. Klicka på "Logga in" för att komma åt din dashboard.

​Återställ ditt lösenord och välj en plan

Välj den perfekta planen för din restaurang och håll dig säker

​Om du skulle glömma ditt lösenord gör Fooodis.com det enkelt att återställa ditt konto. Den här delen av guiden visar hur du återställer ditt lösenord genom att ange din registrerade e-postadress och få en återställningslänk.Dessutom kan du, när du är inloggad, utforska olika prenumerationsplaner som passar storleken och omfattningen av din restaurang. Oavsett om du driver ett litet bistro eller en större verksamhet erbjuder Fooodis flexibla planer som är anpassade efter dina specifika behov.Gå enkelt till avsnittet "Prisplaner" i din dashboard för att komma igång.

Återställa ditt lösenord och välja en plan

Hur du återställer ditt lösenord och väljer en plan:

7. Glömt ditt lösenord?
  ○ Om du har glömt ditt lösenord, klicka på länken "Glömt lösenord" som finns under inloggningsformuläret (7).

8. Återställ ditt lösenord
  ○ På sidan för lösenordsåterställning (8), ange den e-postadress du använde för att registrera din restaurang. Du kommer att få ett e-postmeddelande med en länk för att återställa ditt lösenord.

9. Välj en plan för din restaurang
  ○ När du har registrerat dig kan du välja en prenumerationsplan baserat på storleken och behoven för din restaurang. Fooodis erbjuder olika planer, från paket för små restauranger till alternativ för större verksamheter.
  ○ På din dashboard, klicka på "Prisplaner" för att visa och välja den plan som bäst passar din restaurang.

​Vanliga frågor om att registrera din restaurang på Fooodis.com

​Att börja din resa med Fooodis.com är enkelt, men vi förstår att du kan ha några frågor när du registrerar din restaurang. För att göra processen smidigare har vi sammanställt en lista med vanliga frågor (FAQs) som täcker hela registreringsprocessen.Från att fylla i registreringsformuläret till att få dina inloggningsuppgifter och återställa ditt lösenord, svarar dessa FAQs på de vanligaste frågorna för att hjälpa dig att snabbt och effektivt få din restaurang registrerad.

​Hur registrerar jag min restaurang på Fooodis.com?
  • ​ör att registrera din restaurang på Fooodis.com, gå till startsidan och klicka på länken "Registrera" högst upp på sidan. Detta tar dig till registreringsformuläret där du anger ditt restaurangnamn, ägarinformation och kontaktuppgifter.När du skickar in formuläret får du ett e-postmeddelande med dina inloggningsuppgifter. Med dessa kan du komma åt din dashboard och börja hantera din restaurang online.
​Vilken information behöver jag för att registrera min restaurang?
  • ​När du registrerar din restaurang behöver du ange följande uppgifter:
    Ditt restaurangnamn
    Ägarens namn
    E-postadress
    Telefonnummer
    Denna information säkerställer att ditt konto sätts upp korrekt och att du kan ta emot viktig kommunikation, inklusive dina inloggningsuppgifter.
​Hur kommer jag åt min restaurangs dashboard efter registrering?
  • ​När du har registrerat dig får du ett e-postmeddelande med dina inloggningsuppgifter, inklusive ditt användarnamn och ett lösenord.Du kan komma åt din restaurangs dashboard genom att besöka startsidan på Fooodis.com och klicka på "Logga in" i toppmenyn. Använd de uppgifter som skickades till dig via e-post för att logga in och börja hantera din restaurang.
​Vad gör jag om jag glömmer mitt lösenord?
  • –
  • –
    ​Om du glömmer ditt lösenord kan du enkelt återställa det. På inloggningssidan klickar du på "Glömt lösenord" och anger den e-postadress du använde när du registrerade din restaurang.Fooodis.com skickar ett e-postmeddelande med en länk för att återställa ditt lösenord, så att du snabbt kan återfå åtkomst till ditt konto.
​Hur väljer jag en prenumerationsplan för min restaurang?
  • ​Efter att du har loggat in på din restaurangs dashboard kan du utforska de olika prenumerationsplaner som erbjuds av Fooodis.com. Varje plan är utformad för att passa olika restaurangstorlekar och behov, från små matställen till stora verksamheter.Klicka på alternativet "Prisplaner" i din dashboard för att visa och välja den plan som bäst passar din verksamhet.
​Kan jag logga in på mitt restaurangkonto när som helst?
  • ​Ja! När du har registrerat dig kan du logga in när som helst genom att klicka på "Logga in"-knappen på startsidan på Fooodis.com. Använd bara den e-postadress och det lösenord du fick vid registreringen för att komma åt din dashboard och hantera din restaurang.
