Genom att aktivera Stripe erbjuder du dina kunder en smidig och säker betalningsupplevelse direkt via ditt onlinebeställningssystem.Att ställa in ditt betalningssystem är snabbt och enkelt – skapa ett Stripe-konto, ange dina API-nycklar i Fooodis-dashboarden, och börja hantera betalningar direkt. Öka kundnöjdheten samtidigt som du säkerställer snabba och problemfria transaktioner för din restaurang.

Guide för betalningsinställningar för att ta emot onlinebetalningar

Steg-för-steg-guide för att ställa in onlinebetalningar

1. Gå till betalningsinställningarna
  ○ Navigera till sektionen Appar (1) i sidomenyn.
  ○ Scrolla ner till Stripe-konfiguration (2), där du ställer in och hanterar dina onlinebetalningsinställningar.

2. Registrera ett Stripe-konto
  ○ För att aktivera onlinebetalningar behöver du först skapa ett Stripe-konto. Besök Stripes webbplatoch följ dessa steg:
     ○ Gå till Stripes startsida och klicka på Registrera.
     ○ Ange din e-postadress och skapa ett lösenord.
     ○ Verifiera din e-postadress och logga in på ditt nya Stripe-konto.
     ○ Slutför registreringen genom att ange dina företagsuppgifter (namn, adress, skatte-ID) och länka ditt bankkonto för utbetalningar.

3. Aktivera Stripe för betalningar i Fooodis
  ○ Efter att du har registrerat dig hos Stripe, gå tillbaka till din Fooodis-dashboard.
  ○ I sektionen Stripe-konfiguration (3) behöver du ange din Stripe Key och Stripe Secret, som du hittar i ditt Stripe-konto under API-inställningar.

4. Spara din Stripe-konfiguration
  ○ När du har fyllt i Stripe Key och Secret i de angivna fälten, klicka på Spara (4) för att tillämpa dina betalningsinställningar. Detta aktiverar Stripe för att bearbeta betalningar när kunder gör beställningar online via din restaurang.

Varför använda Stripe för onlinebetalningar?
Stripe erbjuder en säker, tillförlitlig och flexibel plattform för att ta emot onlinebetalningar. 

Genom att integrera Stripe med Fooodis säkerställer du att dina kunder kan betala för sina beställningar enkelt och säkert, vilket förbättrar deras upplevelse samtidigt som betalningsprocessen förenklas för din restaurang.

​Vanliga frågor om att ställa in onlinebetalningar

​​Att aktivera onlinebetalningar via Stripe på Fooodis är ett enkelt sätt att förbättra din restaurangs effektivitet och erbjuda en smidig upplevelse för dina kunder. Nedan finns några vanliga frågor som kan hjälpa dig att navigera genom processen för att aktivera och hantera onlinebetalningar via Stripe. Stripe-konto

​Vad är Stripe och varför behöver jag det?

​Stripe är en säker och pålitlig betalningsplattform som gör det möjligt för din restaurang att ta emot betalningar online. Genom att integrera Stripe med Fooodis kan du bearbeta kundbetalningar direkt när de gör beställningar via ditt restaurangs onlinebeställningssystem.

​Hur skapar jag ett Stripe-konto?

​För att skapa ett ​​​​​​Stripe-konto, besök Stripes webbplats och registrera dig med din e-postadress. Du behöver ange dina företagsuppgifter och bankkontoinformation för utbetalningar. När registreringen är klar kan du få åtkomst till dina API-nycklar för integration med Fooodis.

​Var hittar jag mina Stripe API-nycklar?

​Efter att du loggat in på ditt Stripe-konto, gå till sektionen Utvecklare och välj API-nycklar. Där hittar du både Stripe Key och Stripe Secret som du behöver ange i sektionen Stripe-konfiguration på Fooodis.

​Hur aktiverar jag Stripe i Fooodis?

​Gå till sektionen Appar på din Fooodis-dashboard och klicka på Stripe-konfiguration. Ange din Stripe Key och Stripe Secret i de angivna fälten och klicka på Spara för att aktivera Stripe-betalningar för dina onlinebeställningar.

​Är onlinebetalningar säkra?

​Ja, Stripe är en mycket säker betalningsprocessor som använder kryptering och branschstandardiserade säkerhetsåtgärder för att skydda alla transaktioner. Genom att använda Stripe säkerställer du att både din restaurang och dina kunder är skyddade från bedrägerier.

​Hur tar jag emot utbetalningar från Stripe?

​När du har kopplat ditt bankkonto under registreringen på Stripe kommer utbetalningar automatiskt att överföras till ditt bankkonto enligt Stripes utbetalningsschema. Du kan visa och hantera utbetalningar från din Stripe-dashboard.

​Kan jag inaktivera onlinebetalningar efter att jag har aktiverat dem?

​Ja, du kan inaktivera Stripe-betalningar när som helst genom att gå tillbaka till sektionen Stripe-konfiguration i Fooodis och ta bort eller inaktivera dina API-nycklar.

​Stöder Stripe flera valutor?

​Ja, Stripe stöder flera valutor. Du kan ställa in den valuta som din restaurang använder i både Stripe och Fooodis för att säkerställa att dina kunder debiteras korrekt.

​Vilka avgifter tar Stripe ut?

​Stripe tar ut en standardtransaktionsavgift för varje betalning som behandlas. Den exakta avgiften beror på din region och typen av transaktion. Du kan hitta detaljerad information om Stripes prissättning på deras webbplats.

​Kan jag se betalningshistorik i Fooodis?

​Ja, alla betalningar som görs via Stripe registreras i Fooodis-systemet, vilket gör att du kan spåra och granska betalningshistoriken direkt från din Fooodis-dashboard.

​Har Fooodis åtkomst till mitt Stripe-konto och mina betalningar?

​Nej, Fooodis har inte åtkomst till ditt Stripe-konto eller dina betalningar. Integrationen med Stripe är säker, och dina API-nycklar låter Fooodis endast behandla betalningar å dina vägnar. All känslig finansiell data, såsom dina utbetalningar och kontoinformation, hanteras säkert av Stripe. Fooodis har ingen direkt åtkomst till eller kontroll över dina medel eller kontouppgifter.
