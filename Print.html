Connecting with PrintTempl and PrintNode

Automated Receipt Printing

Fooodis integrates seamlessly with PrintTempl and PrintNode to offer automated receipt printing, simplifying restaurant order management. Choose between these two third-party services based on your needs and connect your printers directly to start printing receipts effortlessly. Both services offer comprehensive guides on their websites to help you set up quickly and tailor their plans according to the number of printers you need to connect.

Access Print Configuration in the Fooodis Dashboard

Printing configurations on the Fooodis Dashboard
Log in to Fooodis Dashboard:
  ○ Use your admin credentials to log in.
  ○ Visit the PrintTempl or PrintNode Websites
  ○ Navigate to the Apps section on the left menu.

PrintTempl or PrintNode: 
  ○ Go to PrintTempl or ​PrintNode and sign up for an account.

Install the Client Software:
  ○ Download or install the ​PrintTempl or ​PrintNode client app to your computer. 
  ○ Log in with your PrintTempl credentials.

Generate an API Key ​ ​PrintTempl or ​PrintNode:
  ○ Navigate to the API section and generate an API key.
  ○ Copy the API key for the next step.

printtempl.com printnode.com

Configure PrintTempl or PrintNode in Fooodis

To set up your printing configurations using either PrintTempl or PrintNode within the Fooodis Dashboard, follow these steps.

1. Access the Print Node or PrintTempl Settings
  ○ ​Go to the Fooodis Dashboard.
  ○ ​Navigate to the Apps section from the left-hand menu.

2. Configure API Key and Printer Settings
  ○ ​Click on either Print Node or PrintTempl from the submenu under Apps.
For PrintTempl
  ○ ​Input the PrintTempl Token and corresponding printer IDs:
     ○ ​Main thermal printer ID
     ○ ​Kitchen thermal printer ID
     ○ ​Standard printer ID for A4 invoices.

For PrintNode
  ○ ​Enter your Print Node API Key and the Printer IDs into the appropriate fields:
     ○ ​Main thermal printer ID
     ○ ​Kitchen thermal printer ID
     ○ S​tandard printer ID.


Customizing Print Preferences in Fooodis

3. Choose and Customize Printing Preferences
  ○ ​In both PrintTempl and Print Node, you can set specific conditions for printing different types of orders.
  ○ ​Customize the printing preferences for:
     ○ ​Print A4 Standard Orders
     ○ ​Print on Main Thermal Printer Orders
     ○ ​Print on Kitchen Thermal Printer Orders

For example, you can choose to print upon order receipt or when the order is accepted. Once you've customized your preferences, click on the Save button.

4. Save Your Changes
  ○ ​Ensure you click the Save button to apply all your configurations.

5. Final Checks and Testing
○ ​Test the setup by printing a receipt from a test order.
○ ​Verify the printer connections to ensure that all printers are recognized and functioning as expected.

Automated Receipt Printing FAQs

Automated receipt printing with Fooodis enhances efficiency and order management in your restaurant. Here are answers to common questions:

How do I connect my printers to PrintTempl or PrintNode?

Visit PrintTempl or PrintNode and follow their step-by-step setup guides. Once connected, configure the settings within Fooodis for seamless receipt printing.

Which service should I choose, PrintTempl or PrintNode?

Both offer robust printing solutions. PrintTempl focuses on generating PDFs from any format, while PrintNode provides extensive hardware support and branding options. Choose based on your restaurant's specific needs.

Can I use both services simultaneously?

Yes, you can connect both services and configure different printers for each one according to your needs.

Is there any support available if I face issues?
  • –
  • –
    Both PrintTempl and PrintNode provide comprehensive customer support. Additionally, Fooodis is here to assist you with any integration challenges.
How do I obtain an API key for PrintTempl or PrintNode?

PrintTempl: Log in to your PrintTempl account, navigate to the API section, or generate an API key.
PrintNode: Log in to your PrintNode account, navigate to the API section, or generate an API key.

How do I configure my printers in Fooodis for automated receipt printing?
  • In the Fooodis Dashboard, go to Apps > Print Node.
  • Enter the API key for PrintTempl or PrintNode.
  • Add your printer IDs or configure print preferences.
Can I customize my printing preferences for different orders?

Yes, in the Print Node settings in Fooodis, you can customize your printing preferences for:
A4 Standard Orders
Main Thermal Printer Orders
Kitchen Thermal Printer Orders

What if my printer is not recognized or doesn't print correctly?
  • Ensure your printer is correctly connected or powered on.
  • Verify that your API key or printer IDs are correctly entered in Fooodis.
  • Check the PrintTempl or PrintNode documentation for troubleshooting.
How many printers can I connect using PrintTempl or PrintNode?
  • The number of printers depends on your subscription plan:
  • PrintTempl: Different plans are available, ranging from 5 to unlimited printers.
  • PrintNode: Offers multiple plans to connect or print from various printers.
